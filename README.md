\# GuardianAI Case Manager



GuardianAI is a security-first case management backend designed to demonstrate modern full-stack architecture using Fastify, Prisma, and PostgreSQL.



\## Architecture



\- Fastify (Node.js API)

\- Prisma ORM

\- PostgreSQL (Docker)

\- TypeScript
